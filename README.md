# Lyriel Todd — Architecture, Landscape, and Ecological Design

The digital archive and ongoing design practice of Lyriel Todd, also known as Lyriel Le.

This repository contains the source for a portfolio exploring architecture, landscape, ecology, adaptive reuse, environmental systems, and multispecies design.

Selected work includes architectural projects, landscape studies, drawings, field observations, research, and experimental habitat design.

## Lyriel Todd / Lyriel Le

Lyriel Todd (Lyriel Le) is an architectural designer whose work investigates the relationship between built environments, living systems, water, landscape, human behavior, and nonhuman habitat.

Rather than treating architecture as an isolated object, the work asks how buildings and landscapes can participate in the ecological systems they occupy.
